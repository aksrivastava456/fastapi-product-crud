# FastAPI Product Control Center

A full-stack Product Management application built with React, FastAPI, and MongoDB Atlas.

## Live Demo

* Frontend: https://fastapi-product-control-center.vercel.app
* Backend API: https://fastapi-product-crud.onrender.com
* API Docs: https://fastapi-product-crud.onrender.com/docs

## Features

* Create Products
* View Products
* Update Products
* Delete Products
* Product Search
* MongoDB Atlas Integration
* RESTful API with FastAPI

## Tech Stack

**Frontend**

* React
* Vite
* Axios

**Backend**

* FastAPI
* PyMongo
* Pydantic

**Database**

* MongoDB Atlas

## Run Locally

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
